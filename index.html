Miladycraft Metaverse

Miladycraft is Remilia's proto-metaverse project, a social environment built on Minecraft's infrastructure. Miladycraft is balanced to accommodate any type of player, whether they want to build, explore, decorate or industrialize. Miladycraft is highly optimized and easy to run on most modern computers, and has a powerful official server with native proximity voicechat.

Miladycraft White Paper

Installation instructions:

1. Buy or own a Minecraft Java Edition account.
2. Download and install Prism launcher for your respective platform (or any launcher of your choosing that supports MultiMC packs)
3. Launch Prism, connect your Microsoft/Minecraft account, hit 'Add Instance' and select "Miladycraft_1.6"
4. Join Milady Village Discord at discord.gg/milady so you can link and connect to the offical multiplayer server.
5. Launch the Miladycraft 1.6 instance and join Miladycraft SMP in Multiplayer! You will be prompted to link with Milady Village Discord upon connecting for the first time.
